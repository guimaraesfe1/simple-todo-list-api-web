# To-Do List Api Web

## Pré-requisitos

- python 3.13
- poetry (Um gerenciador de pacotes e ambiente virutal)
- git

## Instalação

- Clone o repositorio
- Copie o arquivo example.env e adicione as variaveis de ambiente
- Crie um ambiente virtual e instale as dependencias
- Execute o servidor
- Acesse localhost:8000/docs

```bash
$ git clone https://github.com/guimaraesfe1/simple-todo-list-api-web.git
$ cd simple-todo-list-api-web
$ cp example.env .env
$ poetry install
$ poetry env activate
$ task dev
```
